# Ranking/Gesamtwertung anlegen

Um ein Ranking aus mehreren Bewerben zu erstellen, muss ein neuer Bewerb mit dem Typ „Ranking“ angelegt werden.

Die Adminoberfläche eines Rankingbewerbs ist analog zu den anderen Bewerbstypen, jedoch müssen einige Dinge nicht eingestellt werden. Bewerbsklassen für die das Ranking erstellt werden soll müssen jedoch angelegt werden. Nennungen bzw. öffnen der Registrierung ist zwar möglich, in den meisten Fällen aber nicht nötig, da ein "Ranking" nur bereits gekletterte Bewerbe berücksichtigt und somit automatisch die Athlet:innen dieser gekletterten Bewerbe im Rankingbewerb importiert werden.

Die Auswerteroberfläche unterscheidet sich deutlich von der der Einzelbewerbe. Die neuen Register sind in den folgenden Subpages erklärt.
