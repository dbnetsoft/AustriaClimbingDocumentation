# Einladungen (Adminoberfläche)

Im Register „Einladungen“ können, vor allem bei Bewerben ohne offene Registrierung, Athlet\*innen Einladungen zu diesem Bewerb geschickt werden. Diese Athlet\*innen erhalten dann per Mail eine Einladung zum Bewerb. Durch Klick auf den Link im Mail können sie sich dann für den Bewerb anmelden.

Im Feld "Einladung erstellen" können einzelne Athlet\*innen eingeladen werden.&#x20;

Im Feld "Kader einladen" werden alle Athlet\*innen des ausgewählten Kaders eingeladen.
